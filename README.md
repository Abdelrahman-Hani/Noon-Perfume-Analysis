# 🌸 Noon Perfumes Sales Analysis

## 📌 Project Overview

This project presents an end-to-end Data Analytics workflow using Python, SQL Server, and Power BI.

The objective is to analyze Noon perfume products to understand pricing strategies, brand performance, customer ratings, and fragrance preferences.

---

## 🛠 Tools Used

- Python (Pandas)
- SQL Server
- Power BI
- PowerPoint

---

## 📂 Dataset

The dataset contains information about perfume products including:

- Brand
- Product Name
- Department
- Fragrance Family
- Concentration
- Prices
- Ratings
- Sellers
- Discount

---

## 🧹 Data Cleaning (Python)

- Removed unnecessary columns
- Checked missing values
- Removed duplicates
- Converted seller ratings count into numeric format
- Created Discount Amount
- Created Discount Percentage

---

## 🗄 SQL Analysis

Performed analysis using:

- Aggregate Functions
- GROUP BY
- HAVING
- CASE
- Window Functions
- Ranking Functions
